Brainf*ck basics:

Language consists of an array of zeros with length of 30000 and a pointer on the first array element. Each cell can have a value between 0 and 255 and overflows.

" > " - move pointer one cell right

" < " - move pointer one cell left

" + " - increment a value on pointer position by 1

" - " - decrement a value on pointer position by 1

" . " - output an ASCII equivalent of current value

" , " - put an ASCII code of symbol on pointer position

" [ " - start of a loop block, repeats until the value in the origin cell is set to 0

" ] " - end of the loop block, breaks a loop if cell is set to 0

Code examples:

Input: ++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.

Output: Hello World!

Input: +++++++++++++[>>+++++<<-]++++++[>++++<-]>++[>.+<-]

Output: ABCDEFGHIJKLMNOPQRSTUVWXYZ

Input: >,4<++++++++[>------<-]>>++++++++[>++++++<-]<[>>+.<<-]>.

Output: 1234
